1. AbsDistinct
Compute number of distinct absolute values of sorted array elements.
Task description

A non-empty zero-indexed array A consisting of N numbers is given. The array is sorted in non-decreasing order. The absolute distinct count of this array is the number of distinct absolute values among the elements of the array.

For example, consider array A such that:

  A[0] = -5    
  A[1] = -3    
  A[2] = -1
  A[3] =  0    
  A[4] =  3    
  A[5] =  6

The absolute distinct count of this array is 5, because there are 5 distinct absolute values among the elements of this array, namely 0, 1, 3, 5 and 6.

that, given a non-empty zero-indexed array A consisting of N numbers, returns absolute distinct count of array A.

  A[0] = -5

  A[2] = -1
  A[3] =  0

Assume that:

  • N is an integer within the range [1..100,000];
  • each element of array A is an integer within the range [−2,147,483,648..2,147,483,647];
  • array A is sorted in non-decreasing order.

Complexity:

Elements of input arrays can be modified.

// you can also use other imports, for example:
// using System.Collections.Generic;

// you can use Console.WriteLine for debugging purposes, e.g.
// Console.WriteLine("this is a debug message");

class Solution {
    public int solution(int[] A) {
        
        int count = 1;            
        int h = 0;              // head
        int t = A.Length - 1;   // tail
                
        if (A[0] == -2147483648) // integer limitation. abs of minimum int cannot exist on C#
        {
            count++;
            h++;
        }
        
        int cur = Math.Max(Math.Abs(A[h]), Math.Abs(A[A.Length - 1]));            
        
        while (h <= t)
        {
            int th = Math.Abs(A[h]);    // temp head
            if (th == cur)
            {
                h++;
                continue;
            }
            
            int tt = Math.Abs(A[t]);    // temp tail
            if (tt == cur)
            {
                t--;
                continue;
            }
            
            if (th >= tt)
            {
                cur = th;
                h++;
            }
            else
            {
                cur = tt;
                t--;
            }
            
            count++;
        }
        
        return count;
    }
}
